# NewInfo

此项目界面制作涉及以下官方文档和三方库：

      1、以ToolBar作为App标题栏
      
      2、为标题栏添加drawer进行menu显示
      3、使用StaggeredGridView制作列表瀑布流效果
      4、使用ListViewAnimations制作列表展示效果(LayoutAnimation可达同样效果)
      网络传输部分技术如下:
      1、使用Volley进行网络传输（对Volley框架进行了修改）
      2、使用Userversal-ImageLoader框架
